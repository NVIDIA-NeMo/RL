# Qwen3.5 397B GRPO (nemo-rl)

MLPerf-Training benchmark: GRPO post-training of Qwen3.5-397B-A17B on agentic
software-engineering tasks (R2E-Gym SWE instances via the OpenHands agent
framework), built on [nemo-rl](https://github.com/NVIDIA-NeMo/RL) with NeMo
Gym environments, Megatron-Core policy training, and vLLM generation on
disaggregated node pools.

See README.md in this directory for hardware requirements, dataset and
checkpoint preparation, container build instructions, and launch commands.
